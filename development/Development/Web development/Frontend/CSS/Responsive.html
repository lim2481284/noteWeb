• CSS for all size - declare from HTML file 
 <link href="assets/css/main.css" rel="stylesheet" type="text/css" media="all"/>



  • CSS for specific size - declare from HTML file 
 <link rel="stylesheet" type="text/css" href="assets/css/responsive/main.css" media="screen and (max-width : 768px)">


  • CSS for specific size - declare from CSS file 
 Screen below 768 px : 

  • @media screen and (max-width : 768px) {    
             // CSS script 
    }

Screen above 768 px :

  • @media="screen and (min-width : 768px) {
             //CSS script 
    }
