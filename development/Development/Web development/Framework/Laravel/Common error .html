Migrate problem 

  • Migrate error - cannot create table problem 

 SQLSTATE[42000]: Syntax error or access violation: 1071 Specified key was too long
 Solution : 
  • Open \app\Providers\AppServiceProvider.php
  •  Add this after: use Illuminate\...   :
    use Schema;
  • Add this line in function boot() :
    \Schema::defaultStringLength(191); 

  • Migrate error - row problem

 columns "created_at" and "updated_at" are not found in this table.
 Solution :
  • Create model for that table 
    php artisan make:model <table_model_name>
  • Open the new created model file and add :  [ add App/<model_name>.php ] 
    public $timestamps = false


Other problem 

    • Cache problem 

    php artisan cache:clear 


    • Database access problem 

    Problem :
    Access denied for user "homestead" ...    even you configure the database .env file 
    Solution :
    Just restart the server by run " php artisan serve " command again 

    Route problem 
    • Route get controller from folder ( Ex : Auth / registerController ) 

    Route::get('forum', 'Auth\registerController@getForum');




    Controller problem 
    • Class 'App\Http\Controllers\API\Controller' not found  ( When you put controller in the folder ) 

    Solution : add ( use App\Http\Controllers\Controller; ) 

    • Field doesn't have a default value ( Even you insert that field data ) 

    Solution : add fillable in model 
    protected $fillable = ['content', 'img_url','title','userID'];


    Model problem 

    • Base table not found ( By default, model will use the model file name as table name ) 

    Solution : set table name for model 
    protected $table = 'centers';

    • Model::all() return nothing even there is record in database 

    Solution : Change the position of the route 





    API problem 

    • Call specific id not working : column 'id' not found 

    Solution : set default primary key for the table 

    protected $primaryKey = 'forumID';

    • MethodNotAllowedHttpException

    Solution : Check your request method is correct or not
