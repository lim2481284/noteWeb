Installation 

  1. Install local server   

  2. Install composer 

  3. Open CMD and download laravel by write the command 
    composer global require "laravel/installer


First time project

 
  1. Create new project 
    composer create-project --prefer-dist laravel/laravel blog "5.5.*"
  2. Run artisan serve and navigate to localhost:8080
    php artisan serve
