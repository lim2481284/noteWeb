  3. Create model for the register form 

    Step 1 : Create model 

    • php artisan make:model Users


  4. Create controller for the register form

     Step 1 : Create controller for register form

    • php artisan make:controller RegisterController

     Step 2 : Add function to controller 

    use App\Users

    public function getForm(){
    return view ('register');
    }

    public function register(Request $request){
    $this->validation($request);
    Users::create($request);
    return redirect(' / ' ) -> with('Status', 'You are registered' );
    }

    public function validation ($request) {
    return $this -> validate($request , [
    'username' => 'required|max:255',
    'email' => 'required|email|unique:users|max:255'
    ]);
    }

     
  5. Create router request to register form 

     Step 1 : Add route request to (routes/web.php)

    Route::GET('register', RegisterController@getForm ) ;
    Route::POST('register', RegisterController@register) ;
