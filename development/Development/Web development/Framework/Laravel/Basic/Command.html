Composer command :

Command Description
composer global require "laravel/installer" To install laravel 
composer create-project --prefer-dist laravel/laravel blog
 To create laravel project 


Artisan command

 Command Description
 php artisan serve To run laravel server 
 php artisan make:migration To create migration file 
 php artisan migrate To run migration file and create table 
 php artisan migrate:rollback To revert migrate command
 php artisan make:controller To create controller file 
 php artisan make:model  To create model file
 php artisan route:list To list all the available route 

Model command  ( Eloquent ORM ) 

Command Description
Model::find ($id) To find the primary key that matched with $id
Model::create($request->all());
 To create Model data 
Model::all() To get all Model data 
Model::where('id','=',$id) To find Model data with custom where clause 
Model::where("id" , $id) -> get()  To get all data from the where clause 
Model::where("id" , $id) -> orderBy('created_at', 'DESC') -> get()  To sort the data by date
