Why relationship

 
  • To provide chaining and querying capabilities.
  • To auto bind foreign key 



Type of relationships

 
  • One to one

    Example : User associated with one Phone 

<?php

namespace App;

use Illuminate\Database\Eloquent\Model;

class User extends Model
{
    
    public function phone()
    {
        return $this->hasOne('App\Phone');
    }
}
